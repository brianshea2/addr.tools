NAME

info.addr.tools - explore identifying information for domain names and IP addresses

SYNOPSIS

info.addr.tools/example.com

info.addr.tools/1.1.1.1

info.addr.tools/2606:4700:4700::1111

info.addr.tools/me

DESCRIPTION

For any given domain name or IP address, info.addr.tools displays relevant DNS records and RDAP registration data.

Domain names and IP addresses within the displayed data are linked to their corresponding info.addr.tools pages.
