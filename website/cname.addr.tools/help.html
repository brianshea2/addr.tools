CNAME.ADDR.TOOLS.

NAME

cname.addr.tools - construct domain name aliases

SYNOPSIS

$ dig example.com.cname.addr.tools ;; ANSWER SECTION: example.com.cname.addr.tools. 300 IN CNAME example.com.

DESCRIPTION

For any domain name name, all DNS queries for name.cname.addr.tools are answered with a CNAME record pointing to name.
