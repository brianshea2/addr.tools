NAME

header-echo.addr.tools - view HTTP request headers, craft HTTP response headers

SYNOPSIS

$ curl -i 'https://header-echo.addr.tools/?response-header=foo' -H 'Request-Header: bar' HTTP/2 200 content-type: text/plain content-length: 119 response-header: foo GET /?response-header=foo HTTP/2.0 host: header-echo.addr.tools user-agent: curl/8.5.0 accept: */* request-header: bar

DESCRIPTION

All HTTP and HTTPS requests sent to header-echo.addr.tools are answered by echoing the request headers to the response body.

Query string parameters from the request URL are used to generate response headers.

The special query string parameter, status-code, sets the HTTP response status code. This must be an integer from 200 to 599.

EXAMPLES

View your client's standard request headers:

header-echo.addr.tools

Generate a redirect:

header-echo.addr.tools/?status-code=302&location=https://example.com

Trigger a login prompt:

header-echo.addr.tools/?status-code=401&www-authenticate=Basic
